# Example Queries

Use these queries to test and interact with the Jasreanna's Place database.

## 1. Fetching the Menu (Grouped by Category)
Retrieves all categories and their available products.

```sql
SELECT 
    c.name AS category_name,
    p.name AS product_name,
    p.price,
    p.description
FROM categories c
JOIN products p ON c.id = p.category_id
WHERE p.deleted_at IS NULL AND p.is_available = 1
ORDER BY c.id, p.name;
```

## 2. Searching for Products by Tag
Find all products tagged as "Vegan".

```sql
SELECT p.name, p.price
FROM products p
JOIN product_tags pt ON p.id = pt.product_id
JOIN tags t ON pt.tag_id = t.id
WHERE t.name = 'Vegan';
```

## 3. Order Summary with Items
Get details for a specific order (e.g., Order #1).

```sql
SELECT 
    o.id AS order_id,
    o.total_amount,
    o.status,
    p.name AS item_name,
    oi.quantity,
    oi.price_at_purchase
FROM orders o
JOIN order_items oi ON o.id = oi.order_id
LEFT JOIN products p ON oi.product_id = p.id
WHERE o.id = 1;
```

## 4. Sales Report (Total by Category)
Calculate total revenue generated by each category.

```sql
SELECT 
    c.name AS category,
    SUM(oi.quantity * oi.price_at_purchase) AS total_revenue
FROM categories c
JOIN products p ON c.id = p.category_id
JOIN order_items oi ON p.id = oi.product_id
GROUP BY c.name;
```

## 5. Soft Deleting a Product
Safely "remove" a product without breaking order history.

```sql
UPDATE products 
SET deleted_at = CURRENT_TIMESTAMP 
WHERE name = 'Signature Latte';
```

## 6. Finding Active Users with No Orders
Identify potential customers who haven't made a purchase yet.

```sql
SELECT email 
FROM users 
WHERE id NOT IN (SELECT DISTINCT user_id FROM orders WHERE user_id IS NOT NULL);
```
